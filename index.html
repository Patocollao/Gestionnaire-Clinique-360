📱 Manuel d'utilisation : Gestionnaire Clinique Réseau 360

Cette application est conçue spécifiquement pour simplifier la gestion administrative et la planification fiscale d'un psychologue travaillant comme travailleur autonome pour une clinique au Québec.

1. Configuration Initiale (⚙️ Config)

Avant de commencer, configurez vos données de base. Ces informations sont utilisées pour générer vos factures :

  • Professionnel : Votre nom complet et titre (ex. Psychologue, OPQ #...).
  • Clinique : Les coordonnées de la clinique à qui vous facturez.
  • Facturation : Le pourcentage retenu par la clinique (par défaut 30%).

2. Onglet Dépenses — Nouveautés

  • Catégories détaillées : L'Internet résidentiel et le cellulaire sont maintenant des catégories séparées. Les logiciels de gestion de patients (Jane, Psylio, etc.) ont leur propre catégorie à 100% déductible.
  • Services groupés : Si vous payez un seul montant couvrant loyer + électricité + Hydro-Québec, utilisez "Services groupés". Entrez le montant total et choisissez le % pro.
  • Sélecteur de % : Pour les dépenses à usage mixte (cellulaire, Internet, loyer, véhicule), un menu déroulant vous permet de choisir précisément le pourcentage professionnel. L'app calcule automatiquement le montant déductible.
  • Repas d'affaires : Limité à 50% selon Revenu Québec — ce plafond est appliqué automatiquement.

💼 Guide des Déductions Fiscales pour Travailleurs Autonomes (Québec)

En tant que travailleur autonome, vous êtes votre propre entreprise. Le gouvernement vous permet de déduire de vos revenus toute dépense engagée dans le but de gagner un revenu professionnel.

Règle d'or : Remboursement vs Déduction

Il est faux de penser que le gouvernement "rembourse" vos dépenses. Le gouvernement accorde une déduction : il soustrait vos dépenses de vos revenus nets. Le résultat est votre Revenu Imposable. Vous récupérez l'équivalent de votre Taux Marginal d'Imposition (TMI). Par exemple, avec un TMI d'environ 36%, chaque dollar de dépense enregistré vous fait économiser 36 cents en impôts.

Exemples pratiques : Comment saisir vos dépenses

1. Dépenses à 100% (Usage exclusivement professionnel)
Exemple : Frais de l'OPQ ($950) ou logiciel Jane App ($80/mois). Vous inscrivez le montant total. Économie d'impôt réelle : environ $342 / $29 par mois.

2. Dépenses mixtes (Usage professionnel et personnel)
Choisissez la catégorie et utilisez le sélecteur de %. L'app calcule le montant déductible automatiquement.

  • Téléphone cellulaire : Facture de $50. Usage pro estimé à 50% → Déductible : $25.
  • Internet résidentiel : Facture de $80. Usage pro (téléthérapie, notes) estimé à 30% → Déductible : $24.
  • Services groupés (Loyer+Élec+Hydro) : $2,200/mois tout compris. Bureau = 10% superficie. Choisissez 10% → Déductible : $220/mois.

☁️ Guide de Connexion : Sauvegarde sur Google Sheets

Connectez l'application à votre propre compte Google pour sauvegarder vos clients et sessions en toute sécurité dans le nuage.

Étape 1 : Créer le fichier de base de données

  • Allez sur Google Sheets et créez un fichier vide.
  • Nommez le premier onglet Clients et ajoutez ces titres en haut (A1 à H1) :
    id | Prénom | Nom | Téléphone | Date_Naissance | Date_Appel | Sexe | Actif
  • Ajoutez un deuxième onglet nommé Sessions et ajoutez ces titres (A1 à J1) :
    id | Date_Session | Client | ClientId | Tarif | Frais | Net | Mois | Facture | Notes
  • Ajoutez un troisième onglet nommé Depenses (sans accent !) et ajoutez ces titres (A1 à G1) :
    id | date | category | pctDed | amountTotal | amountDed | notes

⚠️ Important : si vous aviez déjà un Apps Script déployé

Vous devez créer un nouveau déploiement (pas modifier l'existant) après avoir remplacé le code, sinon les nouvelles actions (getDepenses, addDepense, deleteDepense) ne fonctionneront pas.

Étape 2 : Ajouter le "cerveau" (Apps Script)

  • Dans le menu en haut, cliquez sur Extensions > Apps Script.
  • Effacez le code présent et copiez/collez exactement ce code :
function doGet(e) {
  var action = e.parameter.action;
  var dataStr = e.parameter.data;
  var data = dataStr ? JSON.parse(dataStr) : {};

  if (action === 'ping') return json({status: 'ok'});

  var ss = SpreadsheetApp.getActiveSpreadsheet();

  // ── CLIENTS ──
  if (action === 'getClients')
    return json({data: getSheetData(ss.getSheetByName("Clients"))});

  if (action === 'addClient') {
    var sheet = ss.getSheetByName("Clients");
    var id = new Date().getTime().toString();
    sheet.appendRow([id, data["Prénom"], data["Nom"], data["Téléphone"],
                     data["Date_Naissance"], data["Date_Appel"], data["Sexe"], data["Actif"]]);
    return json({id: id});
  }

  // ── SESSIONS ──
  if (action === 'getSessions')
    return json({data: getSheetData(ss.getSheetByName("Sessions"))});

  if (action === 'addSession') {
    var sheet = ss.getSheetByName("Sessions");
    var id = 's' + new Date().getTime().toString();
    sheet.appendRow([id, data.Date_Session, data.Client, data.ClientId,
                     data.Tarif, data.Frais, data.Net, data.Mois, data.Facture, data.Notes]);
    return json({id: id});
  }

  // ── DÉPENSES ──
  if (action === 'getDepenses')
    return json({data: getSheetData(ss.getSheetByName("Depenses"))});

  if (action === 'addDepense') {
    var sheet = ss.getSheetByName("Depenses");
    var id = 'd' + new Date().getTime().toString();
    sheet.appendRow([id, data.date, data.category,
                     data.pctDed, data.amountTotal, data.amountDed, data.notes]);
    return json({id: id});
  }

  if (action === 'deleteClient') {
    var sheet = ss.getSheetByName("Clients");
    var rows = sheet.getDataRange().getValues();
    // Try match by id first (col A), then by Prénom+Nom (col B+C) if id is empty
    for (var i = 1; i < rows.length; i++) {
      var rowId = String(rows[i][0]).trim();
      var cleanId = String(data.id).replace('_row_','').trim();
      // Match by _row_ index
      if (data.id && String(data.id).startsWith('_row_')) {
        var rowIndex = parseInt(data.id.replace('_row_',''));
        if (i - 1 === rowIndex) { sheet.deleteRow(i + 1); return json({deleted: true}); }
      } else if (rowId !== '' && rowId === String(data.id)) {
        sheet.deleteRow(i + 1); return json({deleted: true});
      }
    }
    return json({deleted: false, error: 'Not found'});
  }

  if (action === 'deleteDepense') {
    var sheet = ss.getSheetByName("Depenses");
    var rows = sheet.getDataRange().getValues();
    for (var i = 1; i < rows.length; i++) {
      if (String(rows[i][0]) === String(data.id)) {
        sheet.deleteRow(i + 1);
        return json({deleted: true});
      }
    }
    return json({deleted: false, error: 'Not found'});
  }

  return json({error: "Action inconnue"});
}

function json(obj) {
  return ContentService.createTextOutput(JSON.stringify(obj))
    .setMimeType(ContentService.MimeType.JSON);
}

function getSheetData(sheet) {
  if (!sheet) return [];
  var data = sheet.getDataRange().getValues();
  if (data.length <= 1) return [];
  var headers = data[0], result = [];
  for (var i = 1; i < data.length; i++) {
    var row = data[i], obj = {};
    for (var j = 0; j < headers.length; j++) obj[headers[j]] = row[j];
    result.push(obj);
  }
  return result;
}
